# IELTS Speaking Topic Bank

This topic bank tracks which topics and questions have been explored with the user. The system selects unexplored or partially-explored topics for Free Chat conversations.

## Exploration Status Codes

| Code | Meaning |
|------|---------|
| not_explored | Never asked |
| in_progress | User started answering but not fully explored |
| completed | User gave a thorough answer to all questions |
| skipped | User didn't want to discuss this topic |

## Question Depth Levels

| Level | Description |
|-------|-------------|
| 1 | Simple preference question |
| 2 | Adds reason/when/how long |
| 3 | Opinion, comparison, or hypothetical |
| 4 | Analysis, reflection, or impact |
| 5 | Abstract thinking, values, or life philosophy |

## Free Chat Selection Strategy

When `/freechat` is triggered:

1. **Check profile.md** for which topics have been explored and to what depth level
2. **Select a topic** that is either:
   - `not_explored` (prioritize)
   - `in_progress` with depth < 3 (complete the conversation)
3. **Start with depth 1** question for new topics
4. **Progress naturally** through deeper questions as conversation allows
5. **Mark topic as completed** when user has answered depth 4-5 questions

## Profile Tracking Format

The system tracks exploration in `profile.md`:

```markdown
## Topic Exploration Status

| Topic | Status | Max Depth Reached | Last Discussed |
|-------|--------|------------------|---------------|
| Favorite Sport | in_progress | 3 | 2026-05-19 |
| Music | not_explored | 0 | - |
```

## Follow-up Question Flow

For each topic, questions should be asked in this order:
1. Start with **depth 1** (simple preference)
2. When user answers well, move to **depth 2** (details)
3. If conversation continues naturally, ask **depth 3** (opinions)
4. For deeper topics, **depth 4** (reflection/values)
5. Reserve **depth 5** for philosophical discussions

The agent should read the conversation and determine:
- Has this topic been explored before?
- What was the last depth level reached?
- Should we continue this topic or move to a new one?
